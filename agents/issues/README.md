# Issues / Risk Backlog

Hier landen Probleme, Risiken und Nacharbeiten, die nicht sofort gelöst werden können.

Konvention:
- Datei pro Issue: `agents/issues/ISSUE-0001-kurzer-titel.md`
- Status: `open | mitigated | accepted | done`
- Severity: `low | medium | high | critical`

Erstellen:
- Manuell aus `agents/issues/TEMPLATE.md`
- Oder per Script: `agents/scripts/new_issue.sh "Kurzer Titel"`
