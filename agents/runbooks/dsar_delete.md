# Runbook: DSAR / Löschung (Art. 15/17 DSGVO)

## Ziel
Auskunft/Löschung nachvollziehbar, fristgerecht und datensparsam abwickeln.

## Schritte
- [ ] Identität verifizieren (angemessen, nicht übererheben)
- [ ] Datenquellen lokalisieren (DB, Object Storage Audio, Logs, Support-Tool)
- [ ] Export erstellen (Auskunft) oder Löschung durchführen (inkl. Backups-Policy dokumentieren)
- [ ] Bestätigung an Nutzer + interne Doku (was/ wann gelöscht)
