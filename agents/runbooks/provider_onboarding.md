# Runbook: Provider hinzufügen (LLM/TTS)

- [ ] ToS/Reselling erlaubt? (Creditsystem)
- [ ] AVV/DPA verfügbar? (Art. 28 DSGVO)
- [ ] Datenregion/Transfers dokumentiert (SCCs falls nötig)
- [ ] Retention/Logging beim Provider verstanden
- [ ] Preis-/Abrechnungsmodell verstanden (tokens/chars/minutes)
- [ ] Technische Limits (max chars/request, rate limits)
- [ ] SSML/Prosody Support (für TTS)
- [ ] Abuse-Risiken (voice cloning, impersonation) + Schutzmaßnahmen
- [ ] Pricing in `planung/credits.md`/Preisplan übernehmen (versioniert)
